## 环境
编程语言：Java
语言依赖： JDBC（用于连接数据库）
数据库： MySQL 8.x
GUI： Java Swing
数据库设计软件： PowerDesigner

## 使用说明
`MySqlScripts` 内存放MySQL数据相关脚本，`PowerDesigner`中存储PowerDesigner的项目文件，包含逻辑视图和物理视图，代码文件在`DBtest`中，建议用IDEA打开。
代码不保证完全正确，但是基本功能算是完善了。如果有用请**star**本项目。
